## Descrição do Projeto

Landing page desenvolvida como Prova N1 da disciplina de Desenvolvimento Web — Uniube.

A página apresenta o sistema **Pethia Software**, com:

- Apresentação do sistema e funcionalidades
- Tabela de planos e preços
- Vídeo de demonstração em HTML5
- Formulário de captação de leads com validação HTML5

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (responsivo)
- Google Fonts (Sora + DM Sans)
- IA ChatGPT utilizada para apoio na criação do vídeo de demonstração
